Photo API Documentation

1. Retrieve all photos

Get a list of all photos in the database.

Endpoint: GET /photos

HTTP Method: GET

Response:

            
                HTTP/1.1 200 OK
                Content-Type: application/json

                [
                    {"id": 1, "title": "Photo 1", "description": "Description 1"},
                    {"id": 2, "title": "Photo 2", "description": "Description 2"},
                    ...
                ]

2. Create a new photo

Add data for a new photo item.

Endpoint: POST /photos

HTTP Method: POST

Request:

            
                Content-Type: application/json

                {
                    "title": "New Photo",
                    "description": "New Description"
                }

Response:

            
                HTTP/1.1 201 Created
                Content-Type: application/json

                {
                    "id": 3,
                    "title": "New Photo",
                    "description": "New Description"
                }

3. Retrieve a specific photo

Get data of a specific photo item by ID.

Endpoint: GET /photos/:id

HTTP Method: GET

Response:

            
                HTTP/1.1 200 OK
                Content-Type: application/json

                {
                    "id": 3,
                    "title": "New Photo",
                    "description": "New Description"
                }

4. Update a specific photo

Change data of a specific photo item by ID.

Endpoint: PUT /photos/:id

HTTP Method: PUT

Request:

            
                Content-Type: application/json

                {
                    "title": "Updated Photo",
                    "description": "Updated Description"
                }

Response:

            
                HTTP/1.1 200 OK
                Content-Type: application/json

                {
                    "id": 3,
                    "title": "Updated Photo",
                    "description": "Updated Description"
                }

5. Remove a specific photo

Remove data of a specific photo item by ID.

Endpoint: DELETE /photos/:id

HTTP Method: DELETE

Response:

            
                HTTP/1.1 204 No Content
